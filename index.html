Item List Page

  • Navigate to any other page using the top right navigation bar
  • Use the search bar to search for items from the database
  • Each item is displayed as a card with a name, image, price and a button to add the item to the cart
  • Click on image to see item description

Add Item Page

  • Navigate to any other page using the top right navigation bar
  • Fill in the Name, Description, and Price fields. Then add the item picture by pasting the URL or clicking to select a file to add from your local computer. Click the Add button to add a new item to the item list
  • Added items will appear below the form, with the most recent on the left
